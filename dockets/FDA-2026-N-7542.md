**Submission and posting are different events.** A submission receipt is not a public
record; only the posted version is the timestamped, publicly retrievable attestation the
comment was filed for. Confirming it requires the submitter's own login and cannot be done
by any seat. **Until that confirmation, this file records the comment as submitted and not
as posted**, and the distinction is kept rather than smoothed because the whole value of a
public comment is that a third party can retrieve it.

## A category note, flagged rather than assumed

nutrition literacy and holds itself to a strict no-health-claim discipline. A regulatory

the only health-category tree available and a calendar has to live somewhere, not because
the subjects belong together. **If this docket is better held in a separate tree, this file
should move rather than accumulate company** — the founder's judgement, flagged here so the
placement is a decision rather than a default.
